Fiche de Réalisation Professionnelle : Gatari Engineering

Statut : En cours de réalisation

1. Contexte du projet

  • Entreprise d'accueil : Gatari Engineering, une jeune entreprise locale spécialisée dans le génie électrique, l'installation de systèmes réseau, et la mise aux normes de tableaux électriques et d'éclairages LED (pour hôtels, restaurants, et particuliers).
  • Problématique : En tant que jeune structure, l'entreprise a un besoin urgent de développer sa visibilité locale pour attirer ses premiers clients et recruter de futurs techniciens électriciens pour faire face à la demande. Elle ne disposait d'aucune présence en ligne.
  • Objectif : Concevoir la présence en ligne de l'entreprise en deux étapes.
    • Phase 1 (Prioritaire) : Création d'un site vitrine grand public pour attirer la clientèle locale et gérer le recrutement.
    • Phase 2 (Évolution future) : Création d'une extension (espace interne sécurisé) pour le suivi des chantiers sur le terrain par les techniciens.

2. Environnement Technologique (Prévu)

  • Architecture : Site Web Responsive (adapté aux ordinateurs et aux smartphones des clients locaux).
  • Technologies Web : HTML5, CSS3 (Framework Bootstrap pour un design moderne et rapide), JavaScript.
  • Back-end & Base de données : PHP et MySQL (pour stocker les offres d'emploi, les candidatures reçues et les messages de contact).
  • Gestion de version : Git et GitHub pour le suivi du projet et du code source.

3. Analyse des Besoins et Fonctionnalités

  • Module "Présentation & Vitrine" :
    • Page d'accueil valorisant l'image de l'entreprise (photos de chantiers en hôtel/restaurant, présentation des prestations).
    • Page de contact avec un formulaire dynamique pour les demandes de devis.
    • Respect des obligations juridiques (page Mentions Légales et conformité RGPD).
  • Module "Recrutement" :
    • Espace affichant les postes ouverts (ex: Recherche Électricien Qualifié H/F).
    • Formulaire permettant aux candidats de postuler en ligne et de déposer leur CV.

4. Démarche de Travail et Méthodologie

  • Phase 1 - Recueil des besoins : Entretien avec le gérant pour lister les mots-clés locaux importants (ex: "Électricien Évry", "Mise aux normes LED") afin de préparer le futur référencement du site (SEO).
  • Phase 2 - Maquettage (Wireframes) : Création des visuels des pages (Accueil, Contact, Recrutement) pour validation par le gérant avant de lancer le développement.
  • Phase 3 - Modélisation de la base de données : Création du schéma conceptuel (MCD) pour la gestion des messages clients et des candidatures (tables Message, Offre, Candidat).
  • Phase 4 - Développement (En cours) : Initialisation du code HTML/CSS de la page d'accueil et configuration de la base de données MySQL locale.

5. Compétences du Bloc E5 Mobilisées

  • Développer la présence en ligne de l'organisation : Participer à la valorisation de l'image de l'entreprise sur les médias numériques, concevoir le référencement des services en ligne et intégrer le cadre juridique (RGPD / Mentions légales).
  • Travailler en mode projet : Analyser les objectifs d'une jeune entreprise, planifier les activités, concevoir des maquettes et rédiger la documentation technique.
