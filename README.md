# Umumararungu Portfolio Website

A simple portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Smooth scrolling navigation
- Contact form with basic validation
- Project showcase section

## Usage

1. Open `index.html` in your web browser to view the website.
2. Customize the content in `index.html` to match your information.
3. Update styles in `css/style.css` for personalization.
4. Add your profile picture to `images/profile.jpg`.
5. Add project images or screenshots to the `images/` folder.

## Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet
- `js/script.js` - JavaScript for interactivity
- `images/` - Folder for images

## Customization

- Replace placeholder text and images with your own content.
- Modify colors and fonts in `css/style.css`.
- Add more sections or features as needed.

## Troubleshooting

- If images don't load, ensure they are in the correct path.
- For local development, you may need a local server if using advanced features.
- Check browser console for any JavaScript errors.
